# Guión - Lección 5: Estilo simple

**Duración estimada:** ~10 min
**Presentador:** Laura Acosta, Victor Jhosef

---

## [0:00 - 0:30] Introducción

"Hola. En esta lección vamos a crear una tarjeta con estilo usando CSS. Aprenderemos a centrar contenido, añadir colores y bordes."

## [0:30 - 2:30] La estructura HTML

"Vamos a crear un `<div>` con la clase `tarjeta`. Dentro pondremos un título `<h2>` y un párrafo `<p>`."

"El `<div>` es un contenedor genérico que nos permite agrupar elementos y aplicarles estilos en conjunto."

## [2:30 - 5:00] Centrar contenido

"Para centrar la tarjeta horizontalmente usamos `margin: auto`. Combinado con un `width` definido, el navegador calcula los márgenes izquierdo y derecho automáticamente."

"Con `text-align: center` centramos el texto que está dentro de la tarjeta."

## [5:00 - 7:00] Colores y bordes

"Añadimos `background-color` para el color de fondo y `color` para el color del texto."

"Con `border` creamos un borde sólido. Por ejemplo: `border: 3px solid black;`."

"Y con `border-radius` redondeamos las esquinas. Un valor de `16px` da esquinas suaves y modernas."

## [7:00 - 8:30] Espaciado

"La propiedad `padding` agrega espacio interno entre el contenido y el borde. Así el texto no queda pegado al borde de la tarjeta."

## [8:30 - 9:30] Demo práctica

"Abran el enlace en los recursos. En el archivo CSS, dentro de `.tarjeta { }`, agreguen las propiedades que hemos visto. Prueben diferentes colores y tamaños."

## [9:30 - 10:00] Cierre

"Con solo unas líneas de CSS podemos crear tarjetas atractivas. Este patrón se usa mucho en páginas web modernas. En la siguiente lección construiremos una página completa."
