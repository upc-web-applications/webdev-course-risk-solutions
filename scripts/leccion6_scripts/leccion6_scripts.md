# Guión - Lección 6: Creación de una Página Web Básica

**Duración estimada:** ~15 min
**Presentador:** Blancas Chávez, Carlos Franco

---

## [0:00 - 1:00] Introducción

"Hola. Hoy vamos a construir una página web completa de perfil personal. Vamos a combinar todo lo que hemos aprendido: HTML para la estructura y CSS para los estilos."

## [1:00 - 3:00] Estructura HTML

"Vamos a crear una página con las siguientes secciones:"

"- Un `<header>` con foto de perfil, nombre y descripción."
"- Una sección `Sobre mí` con un párrafo personal."
"- Una sección `Mis habilidades` con una lista."
"- Una sección `Mis intereses` con enlaces."
"- Un `<footer>` con derechos de autor."

"Primero escribamos el HTML. Noten cómo usamos etiquetas semánticas como `<header>`, `<section>` y `<footer>` en lugar de solo `<div>`."

## [3:00 - 5:30] Estilos generales

"Ahora vamos al CSS. Primero hacemos un reset: `* { margin: 0; padding: 0; box-sizing: border-box; }` para eliminar los espacios por defecto."

"Luego damos estilo al `body`: fuente Arial, color de fondo gris claro y color de texto oscuro."

## [5:30 - 8:00] Estilos del encabezado

"El `header` tendrá fondo azul, texto blanco y centrado. La imagen de perfil la hacemos circular con `border-radius: 50%`, le damos un borde blanco y un tamaño fijo."

"El título y el párrafo dentro del header se estilizan por separado usando selectores como `header h1` y `header p`."

## [8:00 - 11:00] Estilos de secciones

"Cada sección la limitamos a un ancho máximo de 600px, la centramos con `margin: auto`, le ponemos fondo blanco, padding, bordes redondeados y una sombra suave con `box-shadow`."

"Los subtítulos `h2` los coloreamos de azul para mantener consistencia."

## [11:00 - 13:00] Listas y enlaces

"Las listas las indentamos con `padding-left`. Los enlaces los estilizamos sin subrayado y con color azul. Al pasar el mouse (`:hover`), aparece el subrayado para indicar que es clickeable."

## [13:00 - 14:00] Footer

"El pie de página sigue el mismo estilo del header: fondo azul, texto blanco y centrado."

## [14:00 - 15:00] Cierre y personalización

"Esta es su página de perfil personal. Pueden cambiar la foto, los colores, añadir más secciones. Experimenten con lo aprendido."

"En la siguiente lección veremos recomendaciones y errores comunes."
