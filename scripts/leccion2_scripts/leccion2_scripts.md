# Guión - Lección 2: Estructura HTML Básica

**Duración estimada:** ~8 min
**Presentador:** Aponte Pablo, Isabel Luisa

---

## [0:00 - 0:30] Introducción

"Hola. En esta lección vamos a aprender la estructura básica de un documento HTML."

## [0:30 - 2:00] Etiquetas y elementos

"HTML funciona con etiquetas. Una etiqueta se escribe entre `< >`. La mayoría vienen en pares: una de apertura y una de cierre con `/`. Por ejemplo: `<h1>Hola</h1>`."

"A esto se le llama un elemento HTML: la etiqueta de apertura, el contenido y la etiqueta de cierre."

## [2:00 - 4:00] Estructura de un documento

"Todo documento HTML tiene esta estructura base:"

"`<!DOCTYPE html>` — le dice al navegador que es un documento HTML moderno."
"`<html>` — el elemento raíz que envuelve todo."
"`<head>` — contiene información sobre la página (título, configuración)."
"`<body>` — contiene todo el contenido visible."

## [4:00 - 6:00] Atributos

"Las etiquetas pueden tener atributos que dan información adicional. Por ejemplo, en `<html lang="es">`, `lang` es un atributo que indica el idioma."

"También los usamos en imágenes y enlaces como veremos más adelante."

## [6:00 - 7:30] Demo en CodePen

"Abran el editor que dejamos en los enlaces. Escriban la estructura básica: `<!DOCTYPE html>`, `<html>`, `<head>` con `<title>`, y `<body>` con un `<h1>` y un `<p>`."

"Vean cómo el navegador interpreta el código y muestra el contenido."

## [7:30 - 8:00] Cierre

"Ya saben cómo estructurar un documento HTML. En la siguiente lección veremos más etiquetas útiles."
