# Guión - Lección 3: Elementos HTML Comunes

**Duración estimada:** ~10 min
**Presentador:** Aponte Pablo, Isabel Luisa

---

## [0:00 - 0:30] Introducción

"Hola. Hoy vamos a ver los elementos HTML más comunes que usarán en sus páginas web."

## [0:30 - 2:00] Encabezados y párrafos

"Los encabezados van de `<h1>` (el más importante) a `<h6>` (el menos importante). Úsenlos para jerarquizar el contenido."

"Los párrafos se escriben con `<p>`. Pueden dar formato con `<strong>` para negrita y `<em>` para cursiva."

## [2:00 - 4:00] Listas

"Las listas desordenadas usan `<ul>` y cada elemento va dentro de `<li>`. Úsenlas cuando el orden no importe."

"Las listas ordenadas usan `<ol>` y automáticamente numeran los elementos. Úsenlas cuando el orden sí importe, como pasos a seguir."

## [4:00 - 6:00] Enlaces

"Los enlaces se crean con `<a>` y el atributo `href`. Por ejemplo: `<a href="https://google.com">Ir a Google</a>`."

"Si queremos que el enlace se abra en otra pestaña, añadimos `target=\"_blank\"`."

## [6:00 - 7:30] Imágenes

"Las imágenes se insertan con `<img>`. Es una etiqueta que se cierra sola. Usa el atributo `src` para la URL y `alt` para una descripción."

"Podemos controlar el tamaño con los atributos `width` y `height`."

## [7:30 - 9:00] Anidamiento

"Podemos combinar elementos. Por ejemplo, una imagen que también es un enlace:"

"`<a href=\"...\"><img src=\"...\"></a>`"

"Esto se llama anidar elementos y es muy común en desarrollo web."

## [9:00 - 10:00] Cierre

"Practiquen creando una página con una lista de intereses, enlaces a sus sitios favoritos y una imagen. En la siguiente lección empezaremos con CSS para dar estilo."
