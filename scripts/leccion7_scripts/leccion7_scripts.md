# Guión - Lección 7: Recomendaciones y Errores Comunes

**Duración estimada:** ~5 min
**Presentador:** Flores Eusebio, Angel Thyago

---

## [0:00 - 0:30] Introducción

"Hola. Llegamos a la última lección del curso. Aquí veremos recomendaciones y errores comunes para que sigan mejorando."

## [0:30 - 2:00] Escribir código ordenado

"Mantengan su código limpio y bien indentado. Usen espacios o tabs de forma consistente. El código ordenado es más fácil de leer, entender y corregir."

"Usen nombres de clases claros en CSS. En lugar de `c1` o `estilo`, usen nombres como `tarjeta`, `menu-principal` o `perfil-usuario`."

## [2:00 - 3:30] Errores comunes

"Error 1: Olvidar cerrar etiquetas. Siempre verifiquen que toda etiqueta de apertura tenga su cierre."

"Error 2: Escribir mal las propiedades CSS. Revisen la ortografía: es `background-color`, no `backround-color`."

"Error 3: No enlazar correctamente el CSS. Verifiquen que la ruta en `<link>` sea la correcta."

"Error 4: Usar clases sin definirlas en CSS. Si ponen `class=\"tarjeta\"`, asegúrense de que `.tarjeta { }` exista en el CSS."

## [3:30 - 4:30] Prueben paso a paso

"No escriban todo el código de una sola vez. Agreguen un cambio, prueben, vean el resultado, y luego agreguen más."

"Si algo no funciona, revisen la consola del navegador. En Chrome pueden abrirla con F12 y ver los errores."

## [4:30 - 5:00] Despedida

"Han completado el curso. Ya saben los fundamentos de HTML y CSS. Pueden crear páginas web básicas desde cero. Sigan practicando y explorando."

"¡Gracias por acompañarnos y felicitaciones por llegar hasta aquí!"
