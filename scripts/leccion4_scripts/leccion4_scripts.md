# Guión - Lección 4: Introducción a CSS

**Duración estimada:** ~12 min
**Presentador:** Laura Acosta, Victor Jhosef

---

## [0:00 - 1:00] Introducción

"Hola. Hasta ahora hemos visto HTML, que define la estructura. Pero las páginas web también necesitan verse bien. Para eso usamos CSS."

## [1:00 - 3:00] ¿Qué es CSS?

"CSS significa Cascading Style Sheets. Con CSS podemos cambiar colores, fuentes, tamaños y la distribución de los elementos."

"El CSS se puede escribir directamente en el HTML, pero lo mejor es crear un archivo separado y enlazarlo."

## [3:00 - 5:00] Cómo enlazar CSS

"En el `<head>` de nuestro HTML añadimos: `<link rel=\"stylesheet\" href=\"estilos.css\">`."

"Esto conecta el archivo HTML con el archivo CSS. Ahí podemos escribir todos nuestros estilos."

## [5:00 - 7:30] Selectores básicos

"Para aplicar estilos necesitamos seleccionar elementos. Hay varias formas:"

"- Selector de elemento: `body { }` aplica a todas las etiquetas `body`."
"- Selector de clase: `.caja { }` aplica a todos los elementos con `class=\"caja\"`."

"Las propiedades se escriben con el nombre, dos puntos, el valor, y punto y coma. Ejemplo: `color: red;`."

## [7:30 - 10:00] Propiedades comunes

"Veamos algunas propiedades:"

"`background-color`: color de fondo."
"`color`: color del texto."
"`font-family`: tipo de letra."
"`font-size`: tamaño del texto."
"`text-align`: alineación del texto (left, center, right)."
"`padding`: espacio interno."
"`margin`: espacio externo. Con `margin: auto` centramos un elemento."
"`border-radius`: bordes redondeados."

## [10:00 - 11:30] Demo práctica

"Abran el enlace de OnlineGDB. Verán un HTML con un título, párrafos y una caja. En el CSS, modifiquen los colores, el tamaño de fuente y observen cómo cambia la página."

"Intentens cambiar el color de fondo del `body` a otro color y el color del `h1`."

## [11:30 - 12:00] Cierre

"CSS es muy poderoso y permite transformar páginas simples en diseños atractivos. En la siguiente lección veremos cómo crear una tarjeta con estilo simple."
