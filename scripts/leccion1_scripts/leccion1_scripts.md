# Guión - Lección 1: Introducción al desarrollo web

**Duración estimada:** ~5 min
**Presentador:** Flores Eusebio, Angel Thyago

---

## [0:00 - 0:30] Introducción

"Hola, bienvenidos al curso de Fundamentos de Desarrollo Web. Hoy vamos a aprender qué es un sitio web y cómo está construido."

## [0:30 - 2:00] ¿Qué es un sitio web?

"Un sitio web es un conjunto de páginas que contienen información como texto, imágenes, videos y enlaces. Pero, ¿cómo se construye un sitio web? Se necesita dos ingredientes principales: HTML y CSS."

"HTML es como el esqueleto de una página. Define la estructura: los títulos, párrafos, imágenes, etc. CSS es como la ropa y el maquillaje: le da color, estilo y diseño a la página."

## [2:00 - 4:00] Conociendo CodePen

"Para escribir nuestro código no necesitamos instalar nada. Vamos a usar CodePen, un editor online que nos permite ver los resultados al instante."

"En la descripción del video encontrarás un enlace para abrir el editor. Verás tres paneles: HTML, CSS y Resultado."

## [4:00 - 5:00] Cierre

"Recuerda: HTML = estructura, CSS = estilo. Puedes modificar el código y ver cómo cambia la página en tiempo real. En la siguiente lección empezaremos a escribir HTML."

"¡Nos vemos!"
